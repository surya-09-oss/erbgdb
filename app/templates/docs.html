Cricket Fantasy API

Free, unlimited, self-hosted JSON API for IPL 2026 & International cricket data with Fantasy Points system.

v3.0
Auto-refresh: every 10s Auth: None required Format: JSON Rate limit: Unlimited
Auto-updating cache: All data is cached and automatically refreshed every 10 seconds from source APIs. Fantasy points are auto-updated every 30 seconds for all live matches (IPL + International) — matches are dynamically discovered from Cricbuzz, no manual configuration needed.

GET /api/fantasy/scoring-rules Scoring rules

Returns the complete fantasy scoring rules including batting, bowling, fielding, strike rate, economy rate, and Playing XI bonus.

Scoring Summary

Batting: +1/run, +4 bonus/four, +7 bonus/six, milestones (+4 at 25, +8 at 50, +12 at 75, +16 at 100), duck -2

Strike Rate (min 10 balls): >170: +6, 150-170: +4, 130-150: +2, 60-70: -2, 50-60: -4, <50: -6

Bowling: +25/wicket, +8 LBW/bowled bonus, +12/maiden, haul bonus (+4 for 3W, +8 for 4W, +16 for 5W)

Economy (min 2 overs): <5: +6, 5-6: +4, 6-7: +2, 9-10: -2, 10-11: -4, >11: -6

Fielding: +8/catch, +4 bonus for 3+ catches, +12/stumping, +12/run out direct, +6/run out assist

Extra: +4 Playing XI bonus, -2 duck penalty

Example: 50 runs with 5 fours = 50 (runs) + 20 (5x4 boundary bonus) + 8 (50-run milestone) + 4 (Playing XI) = 82 points

Example Request
curl {{ base_url }}/api/fantasy/scoring-rules

Player Match-Wise Points

How it works: Each match's points start from 0. When you process a match via /api/fantasy/points/{match_id}, each player's points are recorded independently. Old match data is always preserved. You can then query any player's full match history, specific match points, or cumulative totals.

Fantasy Scorecard

Separate from live scores: The fantasy scorecard endpoint returns raw innings data (batting, bowling, fielding) combined with calculated fantasy points. This is different from the live scores endpoints which only show current match state.
GET /api/fantasy/scorecard/{match_id} Full fantasy scorecard

Returns the detailed fantasy scorecard for a match — raw innings data plus calculated fantasy points. Works for both IPL and International matches.

ParameterTypeDescription
match_id required string Cricbuzz match ID (e.g., 149618 for IPL, 122731 for SA vs NZ)
refresh boolean Force refresh from Cricbuzz (default: false)
Example Requests
# IPL match scorecard
curl {{ base_url }}/api/fantasy/scorecard/149618

# SA vs NZ match scorecard
curl {{ base_url }}/api/fantasy/scorecard/122731

# Force refresh
curl {{ base_url }}/api/fantasy/scorecard/122731?refresh=true
Sample Response
{

  "cached": false,
  "data": {
    "match_id": "122731",
    "scorecard": {
      "innings": [
        {
          "batting": [...],

          "extras": {...}
        }
      ],
      "fielding": {...}
    },
    "fantasy_points": [
      {
        "name": "Aiden Markram",

Auto-Update Status

Background auto-update: Fantasy points are automatically refreshed every 30 seconds for all tracked matches. This runs as a background task — no action needed from your side. Use this endpoint to check the status.
GET /api/fantasy/auto-update-status Auto-update task status

Returns the status of the background auto-update task that keeps fantasy points fresh for all tracked matches.

Example Request
curl {{ base_url }}/api/fantasy/auto-update-status
Sample Response
{
  "status": "success",
  "auto_update": {
    "enabled": true,
    "interval_seconds": 30,
    "tracked_matches": ["149618", "122731"],
    "total_tracked": 2,
    "task_running": true
  }
}
