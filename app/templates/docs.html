Cricket API

Free, unlimited, self-hosted JSON API for live cricket scores, IPL data, and more.

v1.0
Auto-refresh: every 10s Auth: None required Format: JSON Rate limit: Unlimited
Auto-updating cache: All data is cached and automatically refreshed every 10 seconds from source APIs. You always get fresh data with zero delay.

General Cricket

GET /api/match-score?id={match_id} Detailed live score

Returns detailed live score for a specific match including batters, bowlers, run rate, and match status.

ParameterTypeDescription
id required string Match ID from Cricbuzz (get from /api/live-matches)
Example Request
curl "{{ base_url }}/api/match-score?id=12345"
Example Response
{
  "status": "success",
  "data": {
    "match_id": "12345",

IPL 2025

GET /api/ipl/live-scores IPL live scores

Returns live scores for all ongoing IPL 2025 matches with the fastest update speed.

Example Request
curl {{ base_url }}/api/ipl/live-scores
GET /api/ipl/schedule Full match schedule

Returns the complete IPL 2025 match schedule with dates, teams, and venues.

Example Request
curl {{ base_url }}/api/ipl/schedule
GET /api/ipl/points-table Current standings

Returns the current IPL 2025 points table with matches, wins, losses, and points for each team.

Example Request
curl {{ base_url }}/api/ipl/points-table
GET /api/ipl/squad/{team_code} Team squad details

Returns the full squad for a specific IPL team including player roles, nationality, and captaincy info.

ParameterTypeDescription
team_code required string Team abbreviation code (see below)
Team Codes
mi Mumbai Indians
csk Chennai Super Kings
rcb Royal Challengers Bengaluru
dc Delhi Capitals
kkr Kolkata Knight Riders
pk Punjab Kings
rr Rajasthan Royals
srh Sunrisers Hyderabad
gt Gujarat Titans
lsg Lucknow Super Giants
Example Request
curl {{ base_url }}/api/ipl/squad/mi
GET /api/ipl/winners Historical IPL winners

Returns the list of all IPL winners from 2008 onwards with runner-up, venue, and margin of victory.

Example Request
curl {{ base_url }}/api/ipl/winners
GET /api/ipl/teams All team codes

Returns the list of all IPL team codes and their full names. Use these codes with the squad endpoint.

Example Request
curl {{ base_url }}/api/ipl/teams
