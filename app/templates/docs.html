Cricket API

Free, unlimited, self-hosted JSON API for live cricket scores, IPL data, and more. Compatible with sanwebinfo/cricket-api.

v2.0
Auto-refresh: every 10s Auth: None required Format: JSON Rate limit: Unlimited
Auto-updating background task: A background task automatically fetches and caches live match data from Cricbuzz every 10 seconds. All endpoints use a 10-second TTL cache, so you always get fresh data with zero delay.

Cricket-API Compatible

Drop-in replacement for sanwebinfo/cricket-api. Same JSON format, auto-updating.

Returns match score in the flat JSON format used by cricket-api. Copy the match ID from Cricbuzz live match URL or from /api/live-matches.

General Cricket

GET /api/live-matches All current live matches

Returns a list of all currently live cricket matches across all formats (T20, ODI, Test) and tournaments.

Example Request
curl {{ base_url }}/api/live-matches
Example Response
{
  "status": "success",
  "cached": true,
  "cache_ttl_seconds": 10,
  "count": 3,
  "matches": [
    {
      "match_id": "12345",
      "title": "India vs Australia, 3rd T20I",
      "status": "In Progress",
      "teams": [
        {"name": "India", "score": "185/4 (20)"},
        {"name": "Australia", "score": "92/3 (12.2)"}
      ]
    }
  ]
}
GET /api/match-score?id={match_id} Detailed live score

Returns detailed live score for a specific match including batters, bowlers, run rate, and match status.

ParameterTypeDescription
id required string Match ID from Cricbuzz (get from /api/live-matches)
Example Request
curl "{{ base_url }}/api/match-score?id=12345"
Example Response
{
  "status": "success",
  "data": {
    "match_id": "12345",
    "title": "India vs Australia, 3rd T20I",
    "status": "India need 42 runs in 30 balls",
    "status_type": "in_progress",
    "live_score": "IND 144/3 (15)",
    "run_rate": "9.60",
    "batters": [
      {"name": "Virat Kohli", "runs": "68", "balls": "42", "strike_rate": "161.90"}

    "bowlers": [
      {"name": "Pat Cummins", "overs": "3", "runs_conceded": "28", "wickets": "1", "economy": "9.33"}

IPL 2025

GET /api/ipl/live-scores IPL live scores

Returns live scores for all ongoing IPL 2025 matches with the fastest update speed.

Example Request
curl {{ base_url }}/api/ipl/live-scores
GET /api/ipl/schedule Full match schedule

Returns the complete IPL 2025 match schedule with dates, teams, and venues.

Example Request
curl {{ base_url }}/api/ipl/schedule
GET /api/ipl/points-table Current standings

Returns the current IPL 2025 points table with matches, wins, losses, and points for each team.

Example Request
curl {{ base_url }}/api/ipl/points-table
GET /api/ipl/squad/{team_code} Team squad details

Returns the full squad for a specific IPL team including player roles, nationality, and captaincy info.

ParameterTypeDescription
team_code required string Team abbreviation code (see below)
Team Codes
mi Mumbai Indians
csk Chennai Super Kings
rcb Royal Challengers Bengaluru
dc Delhi Capitals
kkr Kolkata Knight Riders
pk Punjab Kings
rr Rajasthan Royals
srh Sunrisers Hyderabad
gt Gujarat Titans
lsg Lucknow Super Giants
Example Request
curl {{ base_url }}/api/ipl/squad/mi
GET /api/ipl/winners Historical IPL winners

Returns the list of all IPL winners from 2008 onwards with runner-up, venue, and margin of victory.

Example Request
curl {{ base_url }}/api/ipl/winners
GET /api/ipl/teams All team codes

Returns the list of all IPL team codes and their full names. Use these codes with the squad endpoint.

Example Request
curl {{ base_url }}/api/ipl/teams
